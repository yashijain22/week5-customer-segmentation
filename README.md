# Customer Segmentation using K-Means Clustering and PCA

This project performs customer segmentation using the **Mall Customers dataset**. The goal is to identify groups of customers based on their purchasing behavior and demographic characteristics.

## Objective

To apply unsupervised learning techniques to group customers into meaningful segments that can help businesses design targeted marketing strategies.

## Dataset

The dataset contains information about mall customers including:

- Age
- Annual Income (k$)
- Spending Score (1–100)

These features help analyze customer spending patterns and financial capacity.

## Methods Used

The following techniques were applied in this project:

- Data Exploration and Visualization
- Feature Scaling
- **K-Means Clustering**
- **Elbow Method** to determine optimal number of clusters
- **Principal Component Analysis (PCA)** for cluster visualization

## Key Insights

Customer segmentation helps businesses understand different customer groups based on income and spending behavior. These insights can be used to create personalized marketing strategies, improve customer engagement, and increase business revenue.
